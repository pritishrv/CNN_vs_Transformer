# CNN vs Transformer

This project compares a Convolutional Neural Network (CNN) and a transformer-based vision model on an image classification task.

The main goal is to study how these two model families behave on a relatively simple dataset with around 10 classes. The project is intended as a practical benchmark that covers model training, evaluation, and explainability.

## Project Intent

- identify a suitable image dataset for the study
- train a CNN model for image classification
- train a transformer-based vision model on the same task
- compare model performance using consistent evaluation metrics
- analyse explainability using saliency maps and attention maps

## Expected Outcome

The project should show:
- how a CNN and a transformer differ in learning and performance on the same dataset
- how their predictions can be interpreted through visual explanation methods
- whether one model family offers clearer or more useful explanations than the other in this setting
