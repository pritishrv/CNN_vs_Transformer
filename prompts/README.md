# Prompts Directory

This directory stores selected prompts used during the project.

## Rules
- Do not add every prompt from the chat.
- Add prompts here when they are intentionally written for reuse, documentation, or experiment tracking.
- Prefer one file per prompt or prompt set when that keeps things clear.
- Use descriptive filenames, for example `data-cleaning-plan.md` or `cnn-architecture-ideas.md`.

## Suggested Prompt Template
```md
# Prompt Title

Date: YYYY-MM-DD
Purpose: Short description

## Prompt
...

## Notes
- Why it was used
- Any important outcome
```
